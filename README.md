# Secure API Lab
Безпечне REST API для управління документами з підтримкою аутентифікації та авторизації. API надає функції для створення, читання, оновлення та видалення документів з контролем доступу на основі ролей користувачів. Реалізовано захист від основних веб-вразливостей та підтримку JWT токенів для безпечної автентифікації.

## Швидкий старт
### Встановлення залежностей
```bash
npm install
```
### Запуск сервера
```bash
npm start
```
Сервер запускається на порту 3000 за замовчуванням.

### Запуск тестового клієнта
```bash
npm test
```
